# Verify Greenlane POS

## Project Overview
This project is a cannabis dispensary point-of-sale system designed for faster customer pickup and ID verification.

The system allows customers to preorder products and upload their ID before arriving at the store.

Staff can then verify the customer quickly when they arrive.

## Main Features
- Customer preorder system
- ID upload and verification
- Express pickup check-in
- Staff dashboard
- Customer status tracking
- Menu and specials display

## Tech Stack
- React
- Node.js
- Firebase
- HTML5 QR Scanner
